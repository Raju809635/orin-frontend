# ORIN User Guide (Student + Mentor App)

This guide explains how to use every major feature in the ORIN mobile app from a user point of view.

## 1. Getting Started

### Register
- Open `Register`.
- Enter:
  - Full Name
  - Email
  - Password
  - Role (`Student` or `Mentor`)
- Submit to create account.

### Login
- Open `Login`.
- Enter email + password.
- After login, app opens your main experience based on role.

## 2. Main Navigation

After login, ORIN gives you 2 navigation systems:

### Bottom Navigation (always visible in app flow)
- `My Profile`
- `Domains`
- `Dashboard` (Student or Mentor dashboard based on role)
- `Network`
- `Posts`

The drawer is grouped for clarity:

#### Main

- Settings screen

## 3. Student User Flow

### A. Explore Mentors
1. Go to `Domains`.
2. Pick domain/category.
3. View approved mentors.
4. Open mentor profile.
5. Book a session.

### B. Book Session + Manual Payment
1. In mentor profile, choose details and book.
2. If payment mode is manual, payment panel shows:
   - amount
   - UPI ID
   - QR image (if configured)
3. Upload payment screenshot.
4. Status becomes waiting for verification.
5. Admin verifies payment.
6. Session moves to confirmed.

### C. Track Sessions
- Open `Student Dashboard`:
  - Pending Payments
  - Booking/Session status
  - Legacy requests (if present)
- Continue pending payment later from dashboard.

### D. Daily Career Progress
In student dashboard you can use:
- Daily task list
- Streak/progress style completion
- Reputation score block
- Suggested growth cards

### E. Network and Posts
- `Network`:
  - create posts
  - follow/unfollow users
  - like, comment, share
  - open user profile from feed
- `Posts`:
  - public feed listing
  - profile-aware post interactions

### F. Messaging
- Open `Messages`.
- Chat with relevant users (mentors/students based on available conversations).

## 4. Mentor User Flow

### A. Profile + Pricing
- Open `Mentor Dashboard` > `Profile & Pricing`.
- Set:
  - title, about, experience
  - session price
  - profile fields
- Save profile.

### B. Availability
- Open `Mentor Dashboard` > `Availability`.
- Add weekly slots or date-specific slots.
- Students can book only available slots.

### C. Requests and Sessions
- `Booking Requests`: accept/reject/manage incoming requests.
- `Sessions`: view scheduled/completed sessions and update session progress.

### D. Mentor Growth
- Mentor dashboard includes:
  - activity and engagement widgets
  - reputation summary
  - follow/network actions
  - daily task components

### E. Mentor Payments (Operational View)
- Manual payment verification happens through admin.
- After verification, session is confirmed and visible in mentor flow.

## 5. My Profile (Both Roles)

`My Profile` provides social-style profile management:
- profile photo
- role and basic details
- stats:
  - Posts
  - Followers
  - Following
  - Connections
- reputation card
- lists:
  - Connections
  - Followers
  - Following
- user posts list

Profile actions:
- Edit profile
- Open settings
- Tap users to open public profile

## 6. Public Profile (Other Users)

When you open someone’s public profile:
- view bio/details
- view followers/following counts
- follow/unfollow
- connect/request actions (when available)
- browse their posts

## 7. AI Assistant

Use `AI Assistant` for:
- domain guidance
- roadmap suggestions
- mentorship and session questions
- quick learning support

## 8. Domain Guide

`Domain Guide` gives structured information:
- domains
- sub-domains
- academic/career path references
- guidance for choosing what to learn next

## 9. News & Updates

`News & Updates` includes:
- category tabs:
  - Tech
  - EdTech
  - Govt Exams
  - Scholarships
  - Opportunities
- language selector (multi-language support)
- news cards with image, title, summary, source, read-more link

## 10. Collaborate

`Collaborate` lets users submit collaboration interest to ORIN:
- idea/team details
- contact details
- collaboration intent

## 11. Complaints

`Complaints` lets users report:
- technical issues
- payment/session concerns
- support problems

Each complaint can be submitted with clear details for admin review.

## 12. Notifications

`Notifications` shows important updates such as:
- session/payment status changes
- network-related alerts
- platform alerts and reminders

## 13. Settings and Policy Pages

From `Settings`, users can access:
- app preferences
- profile visibility and options (where available)
- support/legal pages:
  - Help & Support
  - Privacy Policy
  - Terms of Use
  - Mentor Policy
  - About ORIN

## 14. Typical End-to-End Journeys

### Student Journey
1. Register/Login
2. Explore domains
3. Open mentor profile
4. Book + upload payment screenshot
5. Wait for verification
6. Chat + attend confirmed session
7. Post progress in network
8. Build reputation over time

### Mentor Journey
1. Register/Login
2. Complete profile + set pricing
3. Add availability slots
4. Receive and manage requests
5. Conduct sessions
6. Engage in network/posts
7. Grow followers + reputation

## 15. Best Practices for Users

- Keep profile updated (photo, bio, goals, skills).
- Use clear notes during booking.
- Upload valid payment screenshot.
- Check notifications daily.
- Use network posts to show progress and attract opportunities.
