# ORIN User Guide (Current App UI + Features)

This is the user-facing guide for the latest ORIN app experience.

## 1. What ORIN Is

ORIN is a career mentorship and growth app for students and mentors with:
- mentor discovery and booking
- social network style feed
- AI career tools
- community growth modules
- daily quiz and reputation progression

## 2. Login and Roles

### Sign up and login
- Register with `Full Name + Email + Password + Role`.
- Login with `Email + Password`.
- No separate OTP step is required in normal app flow.

### Roles
- `Student`: mentorship, AI tools, network, community, quiz, sessions.
- `Mentor`: requests, sessions, profile/network/posts, mentoring tools.

## 3. Main Navigation

## 3.1 Bottom Navigation (Primary)
The app uses 5 bottom tabs:
1. `Dashboard`
2. `Mentorship`
3. `Network`
4. `AI`
5. `Community`

Notes:
- Network is central in usage and opens the circle activity/feed experience.
- All tabs are designed for full-page flows (not compact overlays).

## 3.2 Side Drawer (Minimal)
Drawer items are intentionally minimal:
- `Domains`
- `Daily Quiz`
- `AI Assistant`
- `News & Updates`
- `Collaborate with ORIN`
- `Settings`

Settings includes policy/support pages (no separate duplicates in drawer).

## 4. Dashboard (Student)

Dashboard is built for quick daily actions.

### Top bar shortcuts
- Profile shortcut
- Notifications
- Messages

### Main sections
- Reputation summary and progress indicators
- Daily Quiz entry and completion state
- Live banners and important updates
- Featured content and discovery cards
- News preview cards
- ORIN Collaborate entry

## 5. Mentorship

Mentorship groups all mentor-learning actions.

### Discovery
- Domains
- Domain Guide
- Mentor profiles
- Verified mentor indicators (where available)

### Session management
- Session History and Notes
- Pending Payments
- Awaiting Verification
- Confirmed Sessions
- Join session actions when meeting link is available

### Payments
- Manual-payment mode supported (where enabled): upload payment proof, then admin verification.

## 6. Network (Social)

Network is ORIN's professional social feed.

### Feed capabilities
- Create post
- Add captions and images (multi-image supported)
- Like, comment, share, and save posts
- Reaction picker on like action
- Follow/unfollow and circle actions
- Open user profile from post header

### Circle terminology
- Circle connections and social interactions use ORIN circle-style naming.

## 7. AI Hub

AI tab opens ORIN career intelligence modules:
- AI Mentor Matching
- AI Skill Gap Analysis
- AI Career Roadmap
- AI Project Idea Generator
- AI Resume Builder
- AI Assistant chat

Each module opens as a dedicated page with details, action controls, and results area.

## 8. Community

Community tab is for growth and engagement:
- Community & Collaboration
- Challenges
- Certifications
- Internship Opportunities
- College Leaderboard
- Knowledge Library
- Reputation and Ranking

Each item opens a dedicated screen.

## 9. Daily Quiz

Daily Quiz focuses on learning continuity:
- domain-based quiz
- score and progress
- XP and streak impact
- one daily completion flow (based on active configuration)

## 10. News & Updates

News module provides career-focused updates:
- Technology
- EdTech
- Government exams
- Scholarships
- Opportunities

Language selector is available. If one source is limited, fallback sources are used based on backend configuration.

## 11. Profile

Profile pages include:
- profile details
- posts count and activity
- followers/following/connections style counters
- edit profile actions
- settings entry

## 12. Notifications and Messages

### Notifications
Used for:
- connection/follow updates
- session/payment status changes
- important app updates

### Messages
Used for:
- student-mentor communication
- network conversations
- session coordination

## 13. Collaborate with ORIN

Collaborate allows users to submit collaboration interest.
Admin can review in the admin dashboard collaborations section.

## 14. Settings and Policies

Settings contains:
- Help & Support
- Complaints
- About ORIN
- Privacy Policy
- Terms of Use
- Mentor Policy
- other app preferences

## 15. Typical User Flows

### Student quick flow
1. Login
2. Dashboard -> Mentorship
3. Choose mentor and book session
4. Complete payment step if required
5. Track status in session sections
6. Join confirmed session
7. Rate mentor and add notes

### Mentor quick flow
1. Login
2. Review requests/sessions
3. Manage availability and profile
4. Coordinate via messages
5. Conduct sessions and track outcomes
